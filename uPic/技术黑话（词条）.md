## Git

**LGTM**

Looks Good To Me「对我来说，还不错」表示认可这次PR，同意merge合并代码到远程仓库

**ASAP**

As Soon As Possible 「尽快」

**ACK**

Acknowledgement 「承认人，确认，同意」ie. agreedlaccepted change

**NACK/NAK**

Negative acknowledgement 「不同意」ie. disagree with change andlor concept

**RFC**

Request For Comments「请求进行讨论」 i.e. 1think this is a good idea, lets discuss 

**WIP**

Work In Progress 「进展中」常见词汇，这里作为 Best Practice（最佳实践） 单独提出来，主要针对改动较多的

加上WIP，表示尚未完成，这样别人可以先 review 已提交的部分

**AFAIK/AFAICT**

As Far As I Know / Can Tell 「据我所知；就我所知』

**IIRC**

If I Recall Correctly 「如果我没有记错的话」

**IANAL**

I am not a lawyer, but I smell licensing issues |-J

**IMO**

In My Opinion 「在我看来」

**TL;DR**

Too Long; Didn't Read 「太长懒得看」README 文档常见。

**PR**

Pull Request 「合并请求」

**CR**

Code Review 「代码审查」

**PTAL**

Please Take A Look.「你来瞅瞅？」用来提示别人来看一下

**TBR**

To Be Reviewed「提示维护者进行 review]

**TBD**

To Be Done(or Defined/ Discussed/Decided/Determined)．「未完成，将被做」根据语境不同意义有

**TBH**

To Be Honest 「老实说」

**atm**

at the moment 「现阶段」
